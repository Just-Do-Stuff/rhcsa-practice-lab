## Task 1

**Exam Category:** Understand and Use Essential Tools 
**Exam Objective:** Access a shell prompt and issue commands with correct syntax 

**Purpose:** 
To practice basic shell access and command syntax using Bash.

### Instructions
1. Open a terminal session on your RHEL system. 
2. Create a directory named `/home/student/practice`.
3. Inside that directory, create a file named `hello.txt`
4. Write the phrase “Welcome to RHCSA Practice” inside `hello.txt`
5. Display the contents of the file on screen.
