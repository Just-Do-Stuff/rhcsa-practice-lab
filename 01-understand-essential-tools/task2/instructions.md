## Task 2

**Exam Category:** Understand and Use Essential Tools 
**Exam Objective:** Use input-output redirection (>, >>, |, 2>, etc.) 

**Purpose:** 
To learn how to redirect command output and errors to files and pipelines.

### Instructions
1. In your `practice` directory, create a new file named `numbers.txt`
2. Add the numbers 1 through 10, each on a new line. 
3. Redirect the output of the `cat` command to a file called `output.txt`
4. Use a pipeline to count how many lines are in `numbers.txt`
5. Redirect the count to a file named `count.log`
