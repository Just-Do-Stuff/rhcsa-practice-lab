## Task 6

**Exam Category:** Understand and Use Essential Tools 

**Exam Objective:** Archive, compress, unpack, and uncompress files using tar, gzip, and bzip2 

---

## INSTRUCTIONS

1. Create a directory named `archive_practice`
2. Inside it, create three text files: `file1.txt`, `file2.txt`, `file3.txt`
3. Archive the directory into a tarball named `practice.tar`
4. Compress the tarball using `gzip` and verify the result.
5. Decompress the `.gz` file, then compress it again using `bzip2`. 
6. Extract the contents of the final archive into `~/extracted`
