## Task 8

**Exam Category:** Understand and Use Essential Tools 

**Exam Objective:** Create, delete, copy, and move files and directories 

---

## INSTRUCTIONS

1. In your home directory, create a new directory named `~/file_mgmt`. 
2. Inside that directory, create three files: `alpha.txt`, `beta.txt`, and `gamma.txt`. 
3. Copy `alpha.txt` to a new subdirectory called `~/file_mgmt/backup`. 
4. Move `beta.txt` into that same backup folder. 
5. Delete `gamma.txt`. 
6. Verify that `alpha.txt` and `beta.txt` exist in the `backup` directory, and `gamma.txt` has been deleted.
