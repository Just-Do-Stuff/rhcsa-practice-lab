# Task 9

**Exam Category:** Understand and Use Essential Tools 
**Exam Objective:** Create hard and soft links 

---

## INSTRUCTIONS

1. Create a directory named `~/link_practice`. 
2. Inside it, create a file named `original.txt` and add the text "Link practice file". 
3. Create a hard link to this file named `hardlink.txt`. 
4. Create a soft link (symbolic link) named `softlink.txt`. 
5. Modify the contents of `original.txt` and verify that changes appear in both linked files. 
6. Verify the type of each link using `ls -l`.
