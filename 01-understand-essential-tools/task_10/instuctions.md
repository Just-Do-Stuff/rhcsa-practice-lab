## Task 10

**Exam Category**: Understand and use essential tools

**Exam Objective**: List, set, and change standard ugo/rwx permissions

## INSTRUCTIONS

1. Create a directory named `~/permissions_practice`. 
2. Inside it, create a file named `access.txt`. 
3. View the current permissions of the file using `ls -l`. 
4. Change the file permissions so that: 
        - The owner has read and write access. 
        - The group has read-only** access. 
        - Others have no access
5. Verify your changes. 
6. Change ownership of the file to another user (if available). 
7. Use `chmod` and `chown` to verify understanding of both commands.
