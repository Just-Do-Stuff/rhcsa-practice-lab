## Task 2

**Exam Category:** Manage software

**Exam Objectives:**
- Configure access to Flatpak repositories
- Install and remove Flatpak software packages

## INSTRUCTIONS

1. Enable the Flatpak framework on your system if not already installed.  
2. Add the official Flatpak remote repository (Flathub).  
3. Search for a Flatpak package (example: `org.gnome.Calculator`).  
4. Install the Flatpak application.  
5. List installed Flatpak applications to verify.  
6. Run the installed application (optional).  
7. Uninstall the Flatpak package and confirm removal.
