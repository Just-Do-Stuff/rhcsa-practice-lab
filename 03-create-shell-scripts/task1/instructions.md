# Category 03: Create Simple Shell Scripts

This category includes two scripting tasks aligned with RHCSA exam objectives.

---

# Task 1: User-Based Script

## Instructions

Write a shell script that:

1. Accepts a username as the first argument (`$1`).
2. Checks if the user exists on the system.
3. If the user exists:
   - Display their home directory.
   - Display their default shell.
4. If the user does not exist:
   - Print a message saying the user does not exist.
5. Exit with:
   - Code `0` if the user exists  
   - Code `1` if the user does not exist
