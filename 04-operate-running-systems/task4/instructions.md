## Exam Category

- **Category:** Operate running systems  
- **Exam Objective:** Interrupt the boot process in order to gain access to a system

---

## Purpose

In this task, you will practice interrupting the boot process using the GRUB menu and resetting the root password.
This is a critical recovery skill and directly aligned with the RHCSA objective for gaining access to a system when normal authentication is not possible.


---

## Instructions 

1. Reboot the system and interrupt the boot process to access the GRUB menu. 
2. Reset the root password to a new value (the exam task will usually specify what the password should be). 
3. Continue the boot process normally and verify that you can log in as root with the new password.
