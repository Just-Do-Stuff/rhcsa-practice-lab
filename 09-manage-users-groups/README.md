## Purpose

This category covers local user and group management.
Tasks focus on safely managing identities, access, and privileges on a Linux system.

---

## What You Will Practice

You will learn how to:

- Create, delete, and modify local user accounts
- Change passwords and configure password aging
- Create, delete, and modify local groups
- Manage group membership
- Configure privileged access using sudo

---

## Quick Disclaimer:

User and privilege misconfiguration can:
- Lock users out of the system
- Break administrative access
- Keep at least one known working admin account
- Avoid modifying the root account unless instructed
