# Contributing Guidelines

Thank you for contributing to the RHCSA Practice Lab!

## General Rules
- Keep all tasks *educational* and *exam-style*.
- Avoid real exam questions or proprietary content.
- Maintain a clear, simple style suitable for beginners.

## How to Contribute
1. Fork the repository.
2. Create a feature branch:
   ```bash
   git checkout -b add-new-task
   ```
3. Add your new task following the format in `/01-understand-essential-tools/`.
4. Commit and push your changes.
5. Open a Pull Request describing your contribution.

## Formatting
- Use Markdown for all task instructions and answers.
- Use consistent indentation and command formatting.
- Do not include output imagesâ€”use command-line examples.
